# Portfolio Carousel Project

This project features a portfolio carousel built with HTML, CSS, and JavaScript. The carousel showcases different items, each with an image, author, title, topic, description, and relevant buttons for further action.

## Overview

The portfolio carousel project demonstrates the following:

1. **HTML Structure**: The main sections include a header with navigation, a carousel container, and a thumbnail section.
2. **CSS Styling**: The project uses custom styling for layout, animation, and responsiveness, including grid-based buttons, animated content, and time-based transitions.
3. **JavaScript Functionality**: The JavaScript code handles the navigation between carousel items, auto-slide functionality, and smooth transitions between items.

## Technologies Used

- **HTML5**: For structuring the web page.
- **CSS3**: For styling and animations.
- **JavaScript**: For interactivity and controlling the carousel.

## Features

### Header

- Contains navigation links to Home, Contacts, and Info.

### Carousel

- **Item Display**: Each item is displayed with an image, author, title, topic, and description.
- **Buttons**: Each item has action buttons (e.g., Email, Telegram, Source code).
- **Thumbnails**: Thumbnail images represent each item and are used for quick navigation.
- **Auto-Slide**: The carousel automatically slides to the next item after a set time interval.
- **Manual Controls**: Users can manually navigate through items using Next and Previous buttons.
- **Animations**: Smooth transitions and animations for both images and text.

### Styling

- **Responsive Design**: The carousel and thumbnails are designed to be responsive to different screen sizes.
- **Custom Fonts**: The project uses the Poppins font from Google Fonts for a modern look.
- **Colors**: A dark theme with contrasting light text for a sleek and professional appearance.
- **Animations**: CSS animations are used for sliding effects and content reveal transitions.

### JavaScript Interactivity

- **Next/Prev Navigation**: JavaScript functions control the carousel's next and previous item transitions.
- **Auto-Play**: The carousel automatically plays, cycling through items with a time delay.
- **Time-Based Animations**: The time bar at the top of the carousel runs to indicate the time left before the next item appears.

## File Structure

- **index.html**: The main HTML file that contains the structure of the web page.
- **style.css**: The CSS file that styles the web page, including the carousel and its animations.

## How to Run

1. **Clone the repository**: Clone this repository to your local machine using `git clone`.
2. **Open the HTML file**: Open `index.html` in your browser to view the portfolio carousel.
3. **Modify as needed**: Feel free to customize the content, styling, or functionality to suit your needs.
