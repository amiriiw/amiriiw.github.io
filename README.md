## How to Use

1. **Viewing the Site**: You can view all the hosted projects using GitHub Pages. To access the full website, visit the [home page](https://amiriiw.github.io/).

2. **Accessing Projects**: All projects are listed as clickable cards on the [projects page](https://amiriiw.github.io/templates/Projects.html). Clicking on a card will take you directly to the corresponding GitHub repository where you can view the code or contribute to the project.
